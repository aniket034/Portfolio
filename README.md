# Research Portfolio Website

A modern, responsive research portfolio website built using HTML, CSS, and JavaScript.

## Overview

This website showcases research experience, academic background, technical skills, projects, and professional achievements in an interactive and visually appealing format.

The design features a dark-themed interface with glassmorphism effects, animated sections, timeline-based layouts, and responsive components optimized for desktop and mobile devices.

---

## Features

### Hero Section
- Animated introduction
- Research interest tags
- Interactive call-to-action buttons
- Smooth scrolling navigation

### About Section
- Professional summary
- Research focus areas
- Achievement highlights
- Interactive information cards

### Academic Journey
- Timeline-based layout
- Animated connectors and milestones
- Academic and professional progression visualization

### Research Experience
- Institution-based experience cards
- Research project summaries
- Organization logos
- Hover animations and transitions

### Research Projects
- Detailed project highlights
- Technology and methodology tags
- Interactive project cards
- Modern glassmorphism design

### Technical Skills
- Categorized skill sections
- Wet lab techniques
- Microbiology expertise

### Contact Section
- Direct email integration
- LinkedIn profile connection
- Responsive contact cards
- Modern UI design
